- 流程控制

- 5种参数输入方式

### 运算符优先级

越上面优先级越高

1. 一元后缀       [ ]    .   ( )  ++   --
2. 一元前缀       ++  --    +   =   ~   !
3. 创建和强制类型转换  new   (type)
4. 倍数  *  /   %
5. 增量  +  -
6. 移位  <<   >>   >>>
7. 关系  <  >   >=  <=  instanceof
8. 等价 ==  !=
9. 与  &
10. 异或  ^
11. 或  |
12. 布尔与  &&
13. 布尔或  ||
14. 条件  ?:
15. 赋值  ＝  +=  -=  *=  /=   %=  >>=  <<=   >>>=   &=   ^=   |=
